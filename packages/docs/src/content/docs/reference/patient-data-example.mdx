This example uses a patient dataset with admission and discharge dates, systolic blood pressure, and weight fields.

It uses the default three-part filter row:

- The left side is the normal `FieldSelect`.
- The middle is a set of custom filters.
- The right side is a custom data input for the selected filter's remaining arguments.

Useful filters to try:

- `admissionDate` days before date between `dischargeDate`, `1`, and `14`
- `dischargeSystolic` less than field `admissionSystolic`
- `dischargeWeight` absolute difference from field is at most `admissionWeight` and `5`
