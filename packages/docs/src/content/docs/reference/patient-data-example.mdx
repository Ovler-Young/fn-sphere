This example uses a patient dataset with admission and discharge dates, systolic blood pressure, and weight fields. It is intended for trying cross-field filters such as date ranges, blood pressure reduction, and absolute weight differences.

Useful filters to try:

- `admissionDate` is between 1 and 14 days before `dischargeDate`
- `dischargeSystolic` is less than `admissionSystolic`
- `dischargeWeight` absolute difference is less than or equal to `admissionWeight` by 5
- `id` contains `P-01`

## Custom Filters and Custom Data Inputs

This version keeps the same patient table and adds patient-specific filters through `defineTypedFn`. The field-pair controls are custom `DataInputViewSpec` components instead of the preset field/expression inputs.

Useful filters to try:

- `admissionDate` is days before selected date, with base `dischargeDate`, min `1`, max `14`
- `dischargeSystolic` is less than selected column, with field `admissionSystolic`
- `dischargeWeight` absolute difference from column is at most, with field `admissionWeight`, threshold `5`
